# P4：接入便利店店主对话切片

Status: done
Milestone: P4
Type: AFK
User stories: 16, 19, 54, 57, 58, 61

## What to build

在共享对话运行期上接入便利店店主。玩家进入便利店后，先看到店主与主角围绕天气、库存和账本的短对话，再进入现有进货、定价和销售流程；对话可以继续或跳过，不改变需求模型、库存、随机种子或行动结果。

人物先使用 32×32 占位小人和统一 fallback。本 issue 不接入角色移动，也不修改便利店商品规则或跨日库存语义。

## Acceptance criteria

- [x] 便利店店主有一段围绕天气、库存和账本的主角短对话。
- [x] 对话发生在经营输入解锁前，关闭前进货、定价、锁定方案和销售按钮均不可推进。
- [x] 便利店复用共享剧情目录、对话运行期、对话框和人物 fallback，不创建便利店专用对话组件。
- [x] 既有教程、商品选择、预算反馈、定价、返回地图、结算和主动放弃行为保持不变。
- [x] 对话继续/跳过输入不会泄漏给进货、定价、锁定方案、销售或返回按钮。
- [x] 对话前后玩家属性、阶段、库存、随机种子和行动结果保持不变。
- [x] 便利店店主素材缺失时显示统一 fallback，不阻断无窗口规则测试。
- [x] 新增台词使用 UTF-8 安全的三行/分页布局并进入字形清单，诊断截图无缺字、重叠或越框。
- [x] 自动化测试覆盖便利店从进入、对话、关闭到经营输入可操作的端到端路径。
