# P3：完成库存清算与七种主结局

Status: ready-for-agent
Milestone: P3
Type: AFK
User stories: 42-50

## What to build

用正式结局判定替换占位主结局。第十日仍完成白天和夜晚行动，随后按集中配置折价清算剩余店铺库存，再按固定优先级从七种结局中选择唯一主结局，并展示最终属性、成长路线和主要判定依据。

## Acceptance criteria

- [ ] 最终库存清算只在第十日完整夜晚活动之后执行一次。
- [ ] 清算将剩余库存按集中配置转为现金，清算后库存和现金保持守恒规则。
- [ ] 第十天评议会文本在结局前或结局页顶部可见，承接餐馆、便利店、图书馆、酒馆和镇长的记忆物件。
- [ ] 结局集合包含小镇明星经营者、冷酷赚钱机器、受欢迎的普通居民、图书馆之星、酒馆传说、平凡小镇新人和经营失败者。
- [ ] 冲突优先级固定为经营失败、综合高成就、单项路线、普通结局。
- [ ] 多个单项路线同时满足时使用集中配置的归一化比较选择最突出指标。
- [ ] 每局只产生一个主结局，不叠加成就或第二结局。
- [ ] 结局页展示最终状态、成长路线和触发主结局的可解释依据。
- [ ] 七种主结局文案遵循 `docs/story/ENDING_AND_ECHOES.md`，不把玩家路线写成好坏评价。
- [ ] 结局状态保存后重新加载仍显示同一结果，不能再次执行库存清算。
- [ ] 使用测试配置覆盖七种结局、优先级冲突、单项并列和普通回退路径。
- [ ] 一条端到端测试从第十日日初推进到唯一主结局。
