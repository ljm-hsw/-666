# P1：扩展为十日周期与占位主结局

Status: ready-for-agent
Milestone: P1
Type: AFK
User stories: 2, 9, 10, 36, 42, 43, 49, 50

## What to build

把单日闭环扩展为固定十日经营周期。每个游戏日使用模拟白天行动和回家夜晚行动推进；第十日仍完成完整白天、夜晚与每日结算，然后进入一个可解释的占位主结局和最终状态总结。

## Acceptance criteria

- [ ] 第一日至第十日均遵循相同的白天、夜晚和每日结算顺序。
- [ ] 第十日不会跳过夜晚活动，也不会进入第十一日。
- [ ] 属性触底不会提前结束游戏，始终保留可完成日程的行动。
- [ ] 第十日结算后只产生一个占位主结局，并展示最终状态和成长路线摘要。
- [ ] 结局之后不能继续提交地点行动或推进天数。
- [ ] 固定种子下，同一十日模拟产生相同当日上下文和结果。
- [ ] 无图形自动化测试完成一条从新游戏到主结局的十日路径。
- [ ] 边界测试覆盖第九日至第十日、第十日结束和非法第十一日推进。
