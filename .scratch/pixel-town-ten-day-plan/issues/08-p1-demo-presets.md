# P1：实现隔离的演示预设加载

Status: ready-for-agent
Milestone: P1
Type: AFK
User stories: 51, 52, 66

## What to build

允许现场演示者通过显式命令行参数加载仓库内的只读演示预设，直接进入指定游戏日、阶段或结局前状态。演示预设不出现在普通菜单，不读取、不覆盖也不更新玩家自动存档。

## Acceptance criteria

- [ ] 合法参数可加载至少一个中期状态和一个结局前状态。
- [ ] 未提供演示参数时，应用只使用正常新游戏或继续游戏流程。
- [ ] 演示预设不出现在普通玩家菜单，也不提供隐藏作弊按钮。
- [ ] 加载和游玩演示预设前后，正式自动存档内容保持不变。
- [ ] 演示预设使用固定种子并能稳定复现当日上下文。
- [ ] 不存在、损坏或版本不兼容的预设会产生明确错误和日志记录。
- [ ] 自动化测试比较演示运行前后的正式存档，证明二者隔离。
- [ ] 构建说明记录演示参数用法，但明确其不属于普通游戏流程。
