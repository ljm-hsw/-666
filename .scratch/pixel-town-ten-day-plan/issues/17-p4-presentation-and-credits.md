# P4：完成像素美术、音频、教程与署名

Status: ready-for-human
Milestone: P4
Type: HITL
User stories: 58, 59, 61, 63, 67, 71, 72

## What to build

用最终获批资源替换开发占位内容，统一小镇地图、地点、状态栏、弹窗、结算和结局的像素表现，加入已确认范围内的背景音乐、关键音效和地点说明，并完成第三方资源署名与许可证交付。

该切片不改变任何地点规则、数值平衡或产品范围。

## Acceptance criteria

- [ ] 全部场景遵循 P0 批准的调色板、像素网格、字体和资源命名规范。
- [ ] P2/P3 期间产生的占位 UI 已统一审查，最终保留、替换或删除的依据记录清楚。
- [ ] 镇长、主角、餐馆老板、便利店店主、图书馆管理员和酒保的小人与统一对话框已替换或明确批准保留，原生尺寸、帧对齐和缺失资源 fallback 通过人工验收。
- [ ] 各地点固定 NPC 的原生像素尺寸、帧对齐、循环待机速度和固定位置通过人工验收；占位程序绘制 fallback 若保留必须有明确人工批准。
- [ ] 固定 NPC 的可见位置、点击热点和标签范围一致；新增、移动、缩放或替换人物视觉时已同步热点并记录检查结果。
- [ ] 简体中文标题、正文、状态值、教程和错误信息在目标窗口中清晰可读。
- [ ] 静态小镇地图热点与场景入口在视觉上可识别，不要求角色自由行走。
- [ ] 每个地点首次进入显示可跳过说明，暂停菜单可重新查看。
- [ ] 最终剧情文本在 960×540 页面上可读，长文本完成分页或压缩，且字体字形覆盖通过。
- [ ] 一首循环背景音乐和地点切换、成功、失败、结算等关键音效正常工作。
- [ ] 全局静音开关覆盖所有音频；音频加载失败时静音降级并记录错误。
- [ ] 所有非原创字体、美术和音频均具有可再分发许可，来源、作者、版本和许可证写入 `CREDITS.md`。
- [ ] 发布所需许可证文本完整，未使用提取自商业游戏的素材。
- [ ] 必要资源清单更新，缺失资源仍进入明确错误页。
- [ ] 演示脚本包含开场、每日提示、第十天评议会和至少一个主结局的讲解路径。
- [ ] imagegen 候选资产若进入最终包，原始生成图、处理后图、hash、用途和人工验收记录完整。
- [ ] 人工视觉、可读性、音频和教程验收结果记录在本 issue 的 `Comments` 中。
