# P3：接入酒馆选择、赌注与夜晚结算

Status: done
Milestone: P3
Type: AFK
User stories: 28, 29, 34, 36

## What to build

建立夜晚酒馆的完整外壳。玩家在夜晚选择回家或进入酒馆；进入酒馆后只能选择一次五子棋或骗子骰子挑战，并在开始前选择合法赌注档位。先使用可控模拟胜负打通挑战结果、赌注、心情、战绩和夜晚阶段结算，为两个真实规则引擎提供稳定接入点。

## Acceptance criteria

- [x] 夜晚地图同时提供回家和酒馆入口，白天不能进入酒馆挑战。
- [x] 回家保持既有恢复和结算路径，不被酒馆实现改变。
- [x] 酒馆只能选择五子棋或骗子骰子中的一个，同一晚不能重复或切换挑战。
- [x] 低、中、高赌注档位由集中配置提供，赌注不能超过当前现金。
- [x] 模拟胜、负、平局分别产生可验证的金钱、心情和酒馆战绩结果；平局退还赌注。
- [x] 酒馆模块只返回行动结果，不直接修改全局状态或推进天数。
- [x] 酒馆行动摘要遵循 `docs/story/STYLE_GUIDE.md`，通过剧情文本入口生成棋盘、骰盅、常客和夜晚不孤单等酒馆回声，不在 UI 绘制代码中硬编码长文案。
- [x] 主动放弃遵循夜晚阶段消耗规则，异常关闭恢复最近阶段边界。
- [x] 规则和集成测试覆盖入口限制、单次挑战、赌注合法性和统一结算。
