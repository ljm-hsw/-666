# P3：接入 1v1 骗子骰子

Status: ready-for-agent
Milestone: P3
Type: AFK
User stories: 32-34, 38, 54, 61

## What to build

把酒馆中的模拟骗子骰子结果替换为可完成的玩家对一个电脑对手的 1v1 淘汰赛。双方持有隐藏骰子，轮流提高叫点或质疑；一点通常作为万能点，叫一点时不作为其他点数的万能点；每轮失败方失去一枚骰子，先失去全部骰子者输。

规则、电脑策略、UI 和酒馆结算必须分离，使规则能够通过固定种子在无窗口测试中复现。

## Acceptance criteria

- [ ] 玩家从酒馆选择骗子骰子后可以完成一场淘汰赛并返回统一结算。
- [ ] 双方只能看到规则允许公开的骰子信息，电脑隐藏骰子不会泄露到玩家 UI。
- [ ] 合法叫点必须严格高于当前叫点，非法叫点被拒绝且不推进回合。
- [ ] 质疑后正确统计普通点数与万能一点，并遵守“叫一点时不万能”的例外。
- [ ] 每轮失败方失去一枚骰子，任一方失去全部骰子时结束比赛。
- [ ] 电脑策略始终提交合法行动并能完成比赛，不要求高级 AI。
- [ ] 相同状态和固定种子产生相同掷骰及电脑行为。
- [ ] 暂停、失焦和最小化不推进回合。
- [ ] 自动化测试覆盖叫点递增、万能点例外、质疑、揭示、失去骰子和最终胜负。
- [ ] 集成测试覆盖酒馆进入、完整比赛、胜负结果和阶段结算。
