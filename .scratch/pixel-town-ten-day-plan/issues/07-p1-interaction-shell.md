# P1：完成窗口、输入、暂停与静音外壳

Status: ready-for-agent
Milestone: P1
Type: AFK
User stories: 56, 57, 59

## What to build

完成所有地点共享的表现外壳：整数倍像素缩放、逻辑坐标输入、鼠标与键盘操作、暂停、窗口失焦冻结和全局静音。该切片只提供稳定的场景与交互能力，不加入新的产品页面或地点规则。

## Acceptance criteria

- [ ] 640×360 逻辑画布在支持的窗口尺寸中使用整数倍缩放与留黑边。
- [ ] 鼠标输入正确转换到逻辑画布坐标，黑边点击不会触发控件。
- [ ] 主要操作可由鼠标完成，常用确认、返回和暂停具有键盘快捷键。
- [ ] 暂停菜单打开时，地点计时、动画推进和规则更新冻结。
- [ ] 窗口失焦或最小化时同样冻结，恢复焦点后继续而不补算失去时间。
- [ ] 全局静音开关即时生效并可持久保存。
- [ ] 交互控件遵循已批准的视觉原型和中文字体规范。
- [ ] 坐标转换、暂停时钟和设置往返具有自动化测试；像素清晰度由人工检查。
