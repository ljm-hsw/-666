# MVP 文本包

本文整理 S 级 MVP 必须实现文本，供 P1 剧情骨架、P2 地点摘要和 P3 结局页引用。文本来源为 [`PIXEL_TOWN_STORY_DESIGN.md`](PIXEL_TOWN_STORY_DESIGN.md)。

## 使用规则

- P1 可以直接使用本文文本，不需要复杂触发条件。
- 文本进入游戏后必须纳入字体字形覆盖检查。
- 运行时文案应从核心剧情层或数据表读取，不在 UI 绘制代码中重复硬编码。
- 文本可按 UI 空间做删减，但不得改变“不评判、不宏大、不羞辱失败”的叙事基调。

## 开场文本

车在小镇入口停下时，路边的旧海报被风掀起一角。镇长拿着钥匙和地图等在站牌旁，地图折痕很深，像被很多人借过。

“先住十天吧。”他说，“餐馆、便利店、图书馆、酒馆都在图上。想去哪里，就从哪里开始。”

## 地点短结果摘要

P1 可先把这些文本作为模拟地点结果摘要。P2/P3 接入真实地点后，应根据地点统计生成或选择更具体的摘要。

| 地点 | 可用摘要 |
| --- | --- |
| 餐馆 | 你把最后一碗汤端到靠窗的位置，老板只点了点头。午饭高峰过去，桌面还热着。围裙沾了点汤汁，但熟客说今天上菜挺快。 |
| 便利店 | 雨伞少了三把，饭团也卖空一排。店主把零钱理进抽屉，账本边角多了一个小勾。汽水卖得慢些，不过夜里来买咖啡的人比预想多。 |
| 图书馆 | 你帮孩子找到了考试要用的书，借书卡又多盖了一个章。管理员把旧地图递给你，说可以先看看，不急着还。书页翻动的声音很轻，窗外的街也慢下来。 |
| 酒馆 | 棋子落下去的声音不大，旁边的人却都看了过来。骰盅打开时，有人笑着拍了拍桌子。你输了这一局，但椅子还给你留着。 |
| 家 | 你烧了热水，把今天的事写进日记。窗灯亮到很晚，街上的声音慢慢退下去。你没有去别的地方，但明天醒来时，体力会好一些。 |

## 第十天评议会开场

第十天晚上，镇长把几张便签摊在桌上。餐馆老板带来一张油点很小的菜单，便利店店主拿着账本，图书馆管理员夹着几张借书卡，酒馆那边有人拎来一副棋子。

镇长清了清嗓子。

“十天不长，”他说，“但也够大家记住一些事了。”

## 七个主结局初版文案

| 结局 | 初版文案 |
| --- | --- |
| 小镇明星经营者 | 十天里，很多地方都见过你。餐馆的午饭高峰、便利店的雨天、图书馆的旧地图、酒馆的夜局，都留下了一点你的痕迹。镇长念到你的名字时，桌边好几个人都抬了头。 |
| 冷酷赚钱机器 | 账本上的数字很漂亮，连便利店店主都多看了两眼。你把进出货、赌注和收益算得清楚，只是有几声招呼落在身后。小镇记得你的效率，也记得你走得很快。 |
| 受欢迎的普通居民 | 你没有做什么夸张的事。只是有时端汤，有时补货，有时帮人找书，有时在酒馆坐一会儿。到第十天，已经有人很自然地跟你打招呼。 |
| 图书馆之星 | 借书卡盒子里多了几张新记录。管理员把旧地图收回去前，指了指边上的空白处，说以后可以把今年的事也写上去。 |
| 酒馆传说 | 酒馆常客说起你时，版本不太一样。有人记得你赢过的棋，有人记得你输后还笑了，还有人只记得你坐在靠窗那张桌边。 |
| 平凡小镇新人 | 十天过去得很快。你走过几条街，认识了几扇门，也错过了一些声音。地图折起来时，纸角比刚来时软了一点。 |
| 经营失败者 | 这十天不算顺。账本空了几格，体力也常常不够。镇长没有把话说重，只把地图推回你面前：小镇还在这里，路也还在。 |

## P1 临时落地策略

P1 接入时可以先采用以下最小策略：

1. 标题页显示开场文本摘要或“新游戏”后的开场卡片。
2. 地图页今日提示使用上表十天每日提示。
3. 模拟地点完成后使用地点短结果摘要替代功能性占位文案。
4. 每日总结页展示地点摘要和一句简短收束。
5. 第十天进入结局前展示评议会开场，或先合并进结局页顶部。
6. 占位结局使用“平凡小镇新人”初版文案，直到 P3 正式结局判定接入。
