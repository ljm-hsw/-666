# 总剧情与叙事框架

本文是《像素小镇：十日经营计划》的剧情总览，用于统一主线、世界观、阶段边界和后续接入位置。原始来源见 [`PIXEL_TOWN_STORY_DESIGN.md`](PIXEL_TOWN_STORY_DESIGN.md) 与同目录 Word 归档。

## 核心叙事问题

> 十天之后，小镇会怎样记住你？

游戏不把玩家推上必须成功、必须拯救小镇的位置。玩家只是以新居民身份临时住进一座有些冷清但仍有生活气的小镇，在十天里通过餐馆、便利店、图书馆、酒馆和回家休息留下自己的生活痕迹。

叙事基调固定为：

- 放松：避免宏大使命和强制拯救叙事。
- 日常：用地图、围裙、雨伞、借书卡、棋子、窗灯、账本、旧海报等具体物件承载情绪。
- 温暖：小镇给玩家回应，而不是审判。
- 不评判：赚钱、安静休息、频繁娱乐或经营失败都只形成生活画像，不被写成道德高低。

## 世界观底盘

### 像素小镇现状

像素小镇曾经更热闹。街口旧海报仍贴着夏日集市的日期；餐馆每天开火但午饭高峰短了些；便利店夜里仍亮着灯；图书馆保留着旧借书卡盒子；酒馆晚上有人下棋，笑声从窗缝里漏出来。

当前小镇不是荒凉废墟，而是一个等待重新被走熟的地方。玩家来到这里不是为了修复所有问题，而是把十天过成自己的样子。

### 小镇十日共创计划

镇长给玩家临时住所、钥匙和地图，邀请玩家以新居民身份住进小镇十天。玩家可以白天去餐馆、便利店或图书馆帮忙，晚上回家休息或去酒馆娱乐。

“共创计划”只是一张看起来正式、内容很朴素的纸：希望有人重新走走这些地方，看看镇上还有什么值得留下。

### 第十天评议会

第十天结束后，小镇举行一次简单评议。镇长、地点负责人、常客或居民把玩家十天里的选择拼成一幅生活画像：去了哪些地方，账本留下什么数字，借书卡多了什么记录，酒馆棋盘旁有没有人提起玩家。

评议会不是成绩单，也不是羞辱失败的场合。它负责把十天收束成“这段日子里，玩家像怎样的人一样生活过”。

### 旧夏日集市暗线

旧夏日集市是后续拓展线索，不进入 MVP 强制目标。

- 餐馆保留着集市用过的大汤锅。
- 便利店仓库里还有旧价签和纸杯。
- 图书馆夹着旧地图，上面圈出过集市摊位。
- 酒馆墙上有褪色照片，照片里的人围着棋桌笑。

这条暗线只提供纵深，不要求玩家完成隐藏任务。玩家可以碰到碎片，也可以只是过完自己的十天。

## 叙事核心机制

剧情串联玩法的核心机制是“选择 -> 回声”。

玩家每天选择地点，不只获得金钱、体力、声望、知识和心情变化，也会在后续文本里留下短小、具体的回声。

| 玩家选择 | 叙事回声方向 |
| --- | --- |
| 多次去餐馆 | 老板少说客套话，熟客记得端汤速度，结算出现热汤、围裙、擦桌等细节。 |
| 多次去便利店 | 店主提到雨伞、汽水和便当销量，晚归的人开始相信玩家的补货判断。 |
| 多次去图书馆 | 管理员递来旧地图，借书卡和泛黄书页成为小镇记忆入口。 |
| 多次去酒馆 | 常客给玩家让座，棋盘和骰子成为夜晚的共同语言。 |
| 多次回家休息 | 文本强调窗灯、热水、日记和第二天，不惩罚安静路线。 |

MVP 先使用静态或少条件文本表达回声。后续条件系统只允许基于地点次数、路线倾向、关键属性和玩法统计扩展，不引入复杂好感度、对话树或任务链。

## 剧情优先级层级

| 层级 | 目标 | 内容 | 实现压力 |
| --- | --- | --- | --- |
| S 级：MVP 必须实现 | 用最少文本让游戏不再像功能拼盘。 | 开场背景、十天每日提示、地点行动结果摘要、每日总结、第十天评议会、主结局文本。 | 不需要复杂 NPC 系统，不需要条件触发系统。 |
| A 级：强烈建议后续实现 | 让地点有“人”。 | 镇长、餐馆老板、便利店店主、图书馆管理员、酒馆老板或常客、主角自我独白。 | 需要少量状态记录，但仍不需要好感度和对话树。 |
| B 级：后期拓展 | 让剧情可以触发，也可以错过。 | 条件事件、NPC 支线碎片、旧夏日集市暗线、结局回忆片段。 | 需要根据地点次数、属性或路线倾向触发。 |
| C 级：远期设计预留 | 只记录方向，不进入当前开发压力。 | 复杂 NPC 支线、多 NPC 交叉事件、隐藏结局、完整夏日集市线。 | 当前不实现。 |

## 与 P0-P4 的关系

| 阶段 | 剧情定位 | 应做内容 |
| --- | --- | --- |
| P0 | 文档和资源前置 | 归档剧情源文档，确认字体/资源能承载中文文本。 |
| P1 | MVP 主线骨架 | 接入 S 级文本：开场、每日提示、地点占位摘要、每日总结、第十天评议会、占位结局文案。 |
| P2 | 白天地点回声 | 餐馆、便利店、图书馆根据真实玩法统计生成具体摘要和地点回声。 |
| P3 | 夜晚与正式结局 | 酒馆回声、第十天评议会、七种主结局正式文本和判定依据。 |
| P4 | 内容扩充和最终润色 | NPC 三段式、旧集市碎片、条件文本、试玩反馈、UI 字数压缩和人工批准。 |

## 硬边界

- 不新增复杂好感度。
- 不新增对话树。
- 不新增任务链。
- 不新增开放世界剧情系统。
- 不改变十日周期。
- 不改变统一行动结果契约。
- 不改变“每局一个主结局”的规则。
- 不通过剧情接入扩大存档格式，除非对应 issue 明确评估并更新存档测试。
