# Triage Labels

工程技能使用五种标准 triage 角色。对于本地 Markdown issue，在文件的 `Status:` 字段中使用下表对应值。

| Matt Pocock 技能角色 | 本仓库状态值 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，Agent 可独立执行 |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 不会处理 |

当技能使用角色描述（例如“AFK-ready”）时，写入对应的本仓库状态值。若以后修改状态词汇，只更新本表右侧的“本仓库状态值”，避免创建含义重复的状态。
